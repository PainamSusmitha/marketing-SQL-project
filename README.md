# 📊 Marketing SQL Project – Digital Campaign Analysis

## 🎯 Objective
To analyze digital marketing campaign performance using SQL and generate business insights based on revenue, leads, orders, CTR, and category performance.

---

## 🛠 Tools Used
- MySQL
- SQL Workbench

---

## 📂 Dataset Overview
The dataset contains marketing campaign data including:
- Campaign Name
- Category
- Impressions
- Clicks
- Leads
- Orders
- Revenue
- Spend
- Campaign Date

---

## 📊 Key Analysis Performed

- Revenue analysis by category
- Top performing campaigns by revenue, leads, and orders
- CTR (Click Through Rate) analysis
- Conversion rate analysis
- Monthly performance trends

---

## 🔍 Key Insights

- Influencer campaigns generated the highest revenue (~21M)
- Instagram Tier 2 generated the highest leads (~10K+)
- YouTube Blogger achieved the highest orders (~1.9K)
- Social and Influencer channels performed best overall

---

## 📈 Skills Demonstrated
- SQL Aggregations (SUM, COUNT, GROUP BY)
- Data Analysis
- Business Insights Generation
- KPI Analysis
- Marketing Performance Analysis

---

## 📌 Conclusion
This project demonstrates how SQL can be used to analyze marketing performance and generate actionable business insights for decision-making.
